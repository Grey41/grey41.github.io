GreyHope Experimental Coding Playground

Welcome to GreyHope

GreyHope is a playground of experimental coding projects. Please visit our projects page to see some complete projects.

Our demos and tutorials are for those who already know the basics of JavaScript and are interested in expanding their JavaScript knowledge,

The website is open source and is hosted on GitHub. Our external website - coffee.greyhope.uk - also contains coding tutorials. Please contact us at info@greyhope.uk if you would like to know more.
