Dweet Runner

This is a program for the Dwitter challenge. It has some useful tools that might come in handy. This program can also be used on mobile devices.

Puck Runner

Use the arrow keys control your man through a procedurally generated maze. Collect the coins to get to the next level and use explosives to get past walls. Beware of the lurking monsters.

Snake

This classic game can be played by two people. The snakes detect obstacles and move toward food. The bigger you get, the more snakes you can surround.

Mandelbrot Viewer

This is the ultimate way to discover the patterns within the mandelbrot set. If you ever find a pattern that you want to share, you can save it as an image.
