Puck Runner

Use the arrow keys control your man through a procedually generated maze. Collect the coins to get to the next level and use explosives to get past walls. Beware of the lurking monsters.

Snake

This classic game can be played by two people. The snakes detect obstacles and move toward food. The bigger you get, the more snakes you can surround.

Mandelbrot Viewer

This is the ultimate way to discover the patterns within the mandelbrot set. If you ever find a pattern that you want to share, you can save it as a png image.

Dweet Runner

This is a program for the Dwitter challenge. The layout is clean and it has some useful tools. This program can also be used on mobile devices.
